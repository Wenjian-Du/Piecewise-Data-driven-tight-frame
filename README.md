## üîß Function Description

### Main Functions
**piecewise_denoising_DDTF**

Main denoising function that implements the complete piecewise DDTF denoising pipeline.

**parset_DDTF_smoothimg**

Parameter settings for natural images with smooth characteristics.

**parset_DDTF_textureimg**

Parameter settings for images with strong features, including montage and emerald parameter configurations.

**Balanced_DDTF_denoising_piecewise**

Piecewise denoising function based on the balanced model.
